    flexbox : 1차원 직선 정렬

  • cf)table : 2차원 정렬

  • main axis(주축), cross axis(교차축. 주축의 수직축)
  • 컨테이너에 설정할 것, 아이템에 설정할 것 다름
    • 컨테이너 속성
      • flex-direction : 주축의 위치(h,v), 주축의 방향(정,역)
        • row(기본값)
        • row-reverse
        • column
        • column-reverse
      • flex-wrap : 강제한줄배치? 여러줄배치?
        • nowrap(기본값) : 강제한줄배치. 아이템들 찌그러짐
        • wrap : 공간 없으면 다음줄로.
        • wrap-reverse : 시작점과 끝점 변경
      • (shorthand)flex-flow : flex-direction flex-wrap 단축속성
    • 아이템 속성
      • order : 아이템 순서. 기본값 0. 음수가능. 부모 컨테이너 flex나 grid 설정되어 있어야
      • flex-grow(늘어남) : 컨테이너 남는 공간 얼만큼씩 나눠가질 건지. 기본값 0
      • flex-shrink(줄어듦) : 기본값 1.(1:1:~으로 줄어들고있)
      • flex-basis : flex-basis 나머지값을 grow,shink가 나눔. 기본값 요소의 크기만큼. 0으로 하면 grow,shink 그 자체로 사용o
      • (shorthand)flex : 아이템을 컨테이너 공간에 맞추기 위해 크기 키우거나 줄이는 속성
        • 값 1개일 때
          • <number> : flex-grow
          • <length>, <%> : flex-basis
          • none, auto, initial
          • 1,2개의 단위 없는 숫자값 쓸 때, flex-basis값이 auto가 아닌 0이 됨
            flex:1과 flex-grow:1은 grow는 같지만 flex-basis는 0과 auto로 다름
